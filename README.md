# Mystery Babylon

Research archive exploring ancient civilizations, secret societies, and astro-theology.

## Live Site

**Domain**: [immortalal.space](https://immortalal.space)

## Content

43 HTML pages covering:
- Ancient civilizations (Egypt, Assyria, Babylon, Persia, Greece, Rome)
- Secret societies (Knights Templar, Freemasons, Illuminati, Skull & Bones)
- Astro-theology (Sun/Moon symbolism)
- Biblical interpretations (Revelation, Armageddon)
- Historical connections and esoteric knowledge

## Structure

- **Entry point**: `mystery.html` (auto-redirects from `/`)
- **Landing page**: `index.html`
- **Cover page**: `cover.html` (Light/Dark choice)
- **Styles**: `styles.css`

## Deployment

- **Hosting**: Netlify (free, auto-deploy on push)
- **Domain**: Cloudflare
- **CDN**: Cloudflare
- **HTTPS**: Automatic via Let's Encrypt

## Updates

```bash
# Edit files locally
git add .
git commit -m "Update [description]"
git push origin main
# Auto-deploys to Netlify in ~1 minute
```
